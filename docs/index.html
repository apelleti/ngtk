11 Commands. Zero Config.

Everything you need to understand an Angular codebase at a glance.

info

Neofetch-style dashboard: versions, artifact counts, standalone/signal/lazy-route health indicators.

env-compare

Compare all environment*.ts files side-by-side and instantly spot missing keys.

component-weight

Rank every component by total file size (TS + HTML + SCSS) to find the heaviest ones first.

dep-map

Map all dependencies categorized as Angular core, ecosystem, or generic third-party.

orphans

Find .ts, .html, and .scss files that nothing imports — safe to delete.

empty-barrel

Detect boilerplate clutter: empty stylesheets, trivial specs, and skeleton services with no methods.

dead-css

Find CSS classes declared in component stylesheets that are never referenced in the template.

route-tree

Print the full route tree as ASCII art, annotated with guards and lazy-loaded markers.

component-catalog

List every component with its selector, standalone status, and file path in a sortable table.

compat-matrix

Cross-check your Angular, TypeScript, RxJS, and Node versions against the official compatibility matrix.

debt-log

Aggregate all TODO/FIXME/HACK comments enriched with git blame age and author — sorted oldest first.

See It In Action

All 11 commands, real output.

Made for CI pipelines.

Every command supports --json. Pipe, filter, fail, report.

🚫 Fail on orphan files

npx ngtk orphans --json \
  | jq 'if length > 0
        then halt_error(1)
        else empty end'

⚠️ Fail on incompatible packages

npx ngtk compat-matrix --json \
  | jq 'if map(select(
          .compatible == false
        )) | length > 0
        then halt_error(1)
        else empty end'

📊 Export debt report

npx ngtk debt-log --json \
  > debt-report.json

# Upload to your dashboard,
# Slack webhook, or S3

🔑 Check env parity before deploy

npx ngtk env-compare --json \
  | jq '.missing
        | if length > 0
          then halt_error(1)
          else empty end'
