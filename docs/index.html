ngtk

Diagnose, analyze, and maintain Angular projects from the terminal.

npx ngtk info

11 Commands. Zero Config.

Everything you need to understand an Angular codebase at a glance.

📊

info

Neofetch-style project dashboard with versions, artifact counts, and health indicators.

🔀

env-compare

Compare environment files side-by-side and spot missing keys across envs.

⚖️

component-weight

Rank components by total file weight — TS + HTML + SCSS combined.

🗺️

dep-map

Map all dependencies categorized as Angular, ecosystem, or generic.

👻

orphans

Find .ts, .html, .scss files not referenced by any import statement.

🗑️

empty-barrel

Detect boilerplate: empty stylesheets, trivial specs, empty services.

✂️

dead-css

Find unused CSS classes in each component's stylesheet.

🌳

route-tree

Print the full route tree as ASCII art with guards and lazy markers.

📖

component-catalog

List all components with selectors, standalone status, and change detection.

✅

compat-matrix

Check Angular / TypeScript / RxJS / Node version compatibility.

📝

debt-log

Aggregate TODO/FIXME/HACK comments with git blame age and author.

See It In Action

Switch between commands to preview their output.

Get Started

Run it once, no install required.

npx ngtk info
